##### IETF
[介绍](https://www.ietf.org/about/introduction/)<br>
The Internet Engineering Task Force (IETF),founded in 1986,is the premiere standards development organization(SDO) for the internet.<br>
互联网工程任务组（英语：Internet Engineering Task Force，缩写：IETF）是一个开放的标准组织，负责开发和推广自愿互联网标准（Internet Standard，英文缩写为STD），特别是构成TCP/IP协议族（TCP/IP）的标准[2]。 它没有正式的会员资格或会员资格要求。 所有参与者和经理都是志愿者，尽管他们的工作通常由雇主或赞助商资助。
